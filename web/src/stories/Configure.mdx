# Evaris AI Design System

Welcome to the Evaris AI component library. This Storybook contains all the UI components used in the Evaris AI dashboard.

## Component Categories

### UI Components
Base UI components from shadcn/ui, customized for Evaris:
- **Button** - Various button styles and sizes
- **Badge** - Status and label badges
- **Card** - Container components
- **Input, Select, Textarea** - Form controls
- **Dialog, Tabs** - Interactive containers
- **Alert, Toast** - Notifications
- And more...

### Dashboard Components
Custom components for the Evaris dashboard:
- **StatCard** - Statistics display with trends
- **ProjectCard** - Project summary cards
- **DatasetCard** - Dataset overview cards
- **RecentActivity** - Activity feed
- **PerformanceChart** - Metric visualization

### Layout Components
- **Logo** - Evaris brand logos and symbols

## Theme Support

Use the theme toggle in the toolbar to switch between light and dark modes. All components support both themes using CSS variables.

## Getting Started

Browse the sidebar to explore individual components. Each component includes:
- Interactive examples
- Props documentation (via autodocs)
- Multiple variants and states

## Development

```bash
# Start Storybook
pnpm storybook

# Build for production
pnpm build-storybook
```
